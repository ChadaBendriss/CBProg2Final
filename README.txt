

Exp1:

This class should be run by specifying:
- Method used for searching: lin or kd
- Parameter value (eps): double number
- Point cloud filename: Point_Cloud1.csv or Point_Cloud_2.csv or Point_Cloud_3.csv 
- Query point of x, y and z-coordinates

Example: java Exp1 lin 0.05 Point_Cloud_1.csv -5.42985 0.80756 -0.39821


Exp2:

The Exp2 class should be run by specifying: 
- Method used for searching: lin or kd
- Parameter value (eps): double number
- Point cloud filename: Point_Cloud1.csv or Point_Cloud_2.csv or Point_Cloud_3.csv 
- Step parameter for the query points: int numbre

Example: java Exp2 kd 0.5 Point_Cloud1.csv 10
